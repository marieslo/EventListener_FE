# Event Listener

## A website for creating and hosting local in-person events
## A full-stack group project

### This is Frontend
### Backend is [here](https://github.com/marieslo/EventListener_BE)

### Technology used:
- Next,
- Nest,
- Typescript,
- Mongo DB,
- Chakra UI,
- Leaflet maps
